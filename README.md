# linux-clock-hud
a tiny little linux clock hud that disapears 1.5 seconds after launch

ment to be launched using a keybind for tiling window managers

INSTALLATION

...

first change directory to the path with the .py file:

$ cd <path/to/.py/file>

then add execute permisions:

$ chmod +x time-overlay.py

finally bind your prefered key to time-overlay.py make sure to use the full path
